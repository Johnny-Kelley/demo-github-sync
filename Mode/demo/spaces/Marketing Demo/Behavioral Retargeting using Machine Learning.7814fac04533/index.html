Behavioral Retargeting using Machine Learning

Which users should we target?


In order to optimize our marketing budget for the rest of the year, we'll have to not only change our Adwords messaging, but will also have to revisit the profile of individuals we have been targeting. Towards that end, we've built a Machine Learning model that identifies the following variables to be the best predictors of whether an Ad viewer will signup or not:


  • Number of employees: We should target users who work companies with less than 5k employees
  • We should keep a gap of at least 16 days between targeting a user again from a previous campaign to the next
  • Users that visit our website spend less than 172 seconds since the time they land on our website. This is crucial because we only have approximately 3 minutes of their attention span to make them Sign up, after which the likelihood of them doing so decreases drastically
