Before Helix, the logic for the main query in this report contained 4 parameters. These parameters were needed due to previous data constraints. While all nine dimensions can be returned using parameters, the result set is not nearly as rich as many elements of the dimensions are not able to be returned. Further, these parameters add to the complexity of the report, and they lead to increased strain on the database as each new set of parameters requires a new query run to be executed.
